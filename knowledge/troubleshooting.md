# คู่มือแก้ปัญหา

Q: ข้อมูลใน Dashboard ไม่แสดงทำอย่างไร?
A: ขั้นที่ 1 ตรวจสอบ Dropdown Year ว่าเลือกถูกปีหรือไม่ ขั้นที่ 2 ลองเลือก All ใน Dropdown ขั้นที่ 3 กด F5 Refresh หน้า

Q: X-bar Chart ไม่มีข้อมูลทำอย่างไร?
A: ตรวจสอบว่า Part No ที่เลือกมีข้อมูล SPC หรือไม่ ลองเปลี่ยน ISC Code หรือเปลี่ยน Year

Q: Cavity Heatmap แสดงสีแดงทั้งหมดหมายความว่าอะไร?
A: หมายความว่า NG เยอะมากในทุก Cavity ต้องแจ้ง Tooling Team และ Production ทันที

Q: ไม่รู้ว่าต้องดู Dashboard ไหนทำอย่างไร?
A: ถ้ามี NG เยอะดู D1 ก่อน ถ้าค่าวัดเกิน Spec ดู D2 ถ้าต้องการดู Cavity ที่มีปัญหาดู D3 ถ้าต้องการย้อนรอย Lot ดู D4

Q: Dashboard เปิดไม่ได้ทำอย่างไร?
A: ตรวจสอบว่าเชื่อมต่อ Network ของบริษัทอยู่หรือไม่ ถ้าไม่ได้ให้ติดต่อ IT Team

Q: ไม่พบข้อมูลที่ต้องการทำอย่างไร?
A: ติดต่อ IT Team หรือ DigitalQC Administrator เพื่อขอความช่วยเหลือ
